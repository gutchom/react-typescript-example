How to develop
--

Run `npm i` at first and then you can use these commands below.

- `npm start`
  - Start building and watching `src/*`
  - Open `http://localhost:3333`
- `npm run testing`
  - Start testing and watching `src/scripts/*`
- `npm run build`
  - Build sorce code with production mode
